# 蓝桥杯 Python 组备考要点

## 竞赛说明

蓝桥杯全国软件和信息技术专业人才大赛，Python 组考查基础语法、算法与简单数据结构。

## 高频考点

1. **输入输出与格式化**：多组输入、`split()`、列表推导
2. **字符串处理**：切片、查找、替换
3. **循环与枚举**：`enumerate`, `zip`
4. **排序与查找**：内置 `sorted`, 二分思想
5. **递归与回溯**：全排列、组合
6. **简单动态规划**：斐波那契、背包入门

## 训练建议

- 每天 1 小时：30 分钟概念 + 30 分钟刷题
- 图解流程图理解循环与递归
- 易错点：边界条件、整数溢出（Python 一般无此问题）、深拷贝

## 推荐学习路径

Python 基础 → 列表字典熟练 → 简单算法 → 真题模拟 → 错题复盘
